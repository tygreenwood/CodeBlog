Assignment 1

October 11, 2021

Going into this project, I had a little bit of background on HTML. Primarily, this background was related to syntax and generic structure. Like the fact that you have headers and paragraphs, and everything appears in the order that it is coded in. What I didn't know as well was the more technical elements, like how to link other pages, or what some of the differences between elements were. Overall, I felt that the assignment wasn't too difficult, and the most trouble I had was figuring out how to actually publish the pages to the internet via the engineering servers. I pretty much was able to figure everything out based on the information I got from lectures, the one external resource I used was the Oregon State page on how to publish pages to the engineering server.

Assignment 2

October 25, 2021

This assignment, when it started as only HTML, was a little overwhelming to begin. My brain was struggling a little to make the website look like the mocks. However, once it game together, it felt very satisfying. I found that breaking this down into smaller pieces was the easiest way to do this assignment. I started by formatting the header, and then the filter sidebar, and lastly the main section, the posts. One of the most useful resources for me was A Complete Guide to Flexboxes, by css-tricks.com. Flexboxes were very useful for laying out my page, and a lot of the formatting for this assignment was done using them. While I learned a lot of good CSS from this assignment, I think the most valuable thing I learned was how to tackle the task of applying CSS to a webpage.

Assignment 4

November 15, 2021

At first, I struggled a lot with this assignment, and I went about doing it in a poor way. I tried to read in all the files and then stick them all in the html file using script and style elements. This caused a lot of errors and didn't load everything properly. I then realized that the way I was supposed to do this was load everything into their own webpage, under their own content type. Doing this immediately made all my code work, and made what seemed like a daunting task (since I basically thought I still had tons of errors and hours worth of work in front of me), a brief, ten minute fix instead.

Assignment 5

December 1, 2021

This may have been the trickiest assignment for me. The actual templatizing of the html was not tricky, as it was basically copying over the html and replacing sections with variables. The hard part of the assignment was figuring out how to run everything in the express server. Certain functions and lines of code shown in the lecture notes didn't work, such as "var exphbs = require('express-handlebars')". In fact, the only way I got it to work was to deconstruct that library and get the engine function from it, "const { engine } = require('express-handlebars')". Besides that, everything else went fairly smooth.
